1、首先注册一个csdn账号，因为我们要用其中的markdown编辑器。
2、打开之后，在右上角进入我的博客。
3、进入之后，选择写博客，然后在左上角选择markdown编辑器
4、写完之后，点击上面的导出，然后选择右面的导出为Markdown文件。
5、导出来之后，在电脑中找到这个.md文件。将其复制下来后，粘贴到你的hexo博客文件夹 中 source 下的 _posts 中。
6、右击选择Git Bash Here,依次输入：
hexo g
hexo s
hexo d
7、 最后浏览器中输入
https://(你的博客名).github.io/ 回车就可以看到你刚刚发的文章了。
